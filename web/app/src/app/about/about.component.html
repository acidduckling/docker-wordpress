About Page

This is a really simple example of configuring the following, all through Docker:

  • Apache - hosting WordPress and static files, plus route configuration
  • Node.js - running a simple Express Web API server - routed via reverse proxy in Apache
  • MySQL - DB for WordPress, and a custom users table generated upon container instantiation
  • MyPHPAdmin - connected to MySQL for web based admin
  • Angular - Static Angular files served through the Apache instance, with rewrite rules for client routing
